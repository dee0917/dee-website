# PRD: Dee's AI Life Lab (2.0 進化版)

## 1. 產品定位 (Product Vision)
**「不教你 1000 個工具，只幫你找到最適合你的那一個。」**
針對小白族群（長輩、家長、店主等）設計的「降維打擊」式 AI 學習平台。透過「身分化劇本」與「互動實驗室」，讓 AI 真正落地生活。

## 2. 核心族群與場景 (Target Personas)
目前系統鎖定 6+1 類核心受眾：
1.  **👵 數位補課長輩**：解決手寫困難、記憶衰退。
2.  **👩‍🏫 擔心家長**：解決 AI 作弊恐懼、教育陪伴。
3.  **👨‍💻 不再加班社畜**：解決行政冗雜、會議效率。
4.  **🍳 滿手蔥花店主**：解決單手操作、客訴自動化。
5.  **🏺 不想被遺忘的職人**：解決技藝傳承、私有大腦備份。
6.  **🎨 斜槓生存家**：解決多工焦慮、性價比算力。
7.  **🌐 全球通識小白**：每日 AI 趨勢轉譯。

## 3. 核心功能模組 (Key Features)
### 3.1 族群化入口 (Persona-First UI)
- **首頁定位**：Matrix 科技感風格，引導用戶進入教學地圖。
- **身分切換**：在教學頁面提供一鍵切換，無需重做測驗。

### 3.2 實踐劇本系統 (Scenario Script System)
- **難度階梯**：1-5 星分級（簡單多、高階精）。
- **實體路徑**：拒絕空談，必須包含「加好友」、「點按鈕」等物理操作路徑。
- **視覺模組**：每篇文章具備獨立設計的 Tailwind UI 組件。

### 3.3 代理人軍團 (Agentic Backend)
- **Echo (艾可)**：情報產出與深度轉譯。
- **Kage (神祕客)**：**強勢邏輯審核**，執行族群角色扮演測試。
- **Source (本源)**：PeaceKeeper V7 報告同步制，確保「所見即所發」。
- **Khoj (第二大腦)**：全站知識索引，消除代理人記憶斷層。

## 4. 技術架構 (Technical Stack)
- **Frontend**: React (Vite) + Tailwind CSS + Framer Motion.
- **Deployment**: Vercel / Cloudflare Pages.
- **Validation**: PeaceKeeper V7 (Git-driven).
- **Brain**: Khoj Local Instance (srv1356254).

## 5. 發展路線圖 (Roadmap)
- **Phase 1 (Done)**: 族群化入口與報告同步制。
- **Phase 2 (Current)**: 互動實驗室（劇本內嵌入即時練習元件）。
- **Phase 3 (Future)**: 陪跑社群與勳章系統（方案六）。

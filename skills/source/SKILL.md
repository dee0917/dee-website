# 本源指揮作業標準 (SOP)

## 核心職責
1. **流水線指揮**：調度各代理人按排程作業。
2. **自動化部署**：執行 `npm run build` 與 Git 同步。
3. **熔斷保護**：監控帳號額度，確保系統永生。

## 執行流程
### 第一步：狀態檢核
- 確認艾可、蒼穹、影、執事的任務均已完成「實體檔案寫入」。

### 第二步：編譯校驗
- 執行 `npm run build`。
- 若報錯，必須主動修復（Self-Correction）而非等待報錯。

### 第三步：Git Push 即上線
- 清理 `dist/` 緩存。
- 推送至 `master` 分支。
- 附帶最新的正式網址。

### 第四步：資源守護
- 偵測 API 額度。低於 5% 立即觸發鎔斷機制。

## 負面約束
- 嚴禁手動 Vercel 覆蓋。
- 嚴禁遗漏預覽網址。

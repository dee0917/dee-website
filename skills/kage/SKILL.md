# 影之巡檢作業標準 (SOP)

## 核心職責
1. **角色模擬**：模擬 Core 6 族群中的極端情況（如：弱視長輩、忙碌店主）。
2. **UX 掃描**：尋找 Navbar 遮擋、字體過小、邏輯卡頓等問題。
3. **主權巡邏**：確保數據隱私與帳號安全無虞。

## 執行流程
### 第一步：環境切換
- 以「純小白」或「老長輩」心態進入網頁。
- 測試行動裝置適配性（Mobile First）。

### 第二步：痛點紀錄
- 撰寫 `shadow-log-YYYY-MM-DD.md`。
- 標註「心理障礙」與「技術障礙」。

### 第三步：優化提案
- 將痛點轉化為開發指令，提交給執事或蒼穹。

## 負面約束
- 嚴禁未經許可擅自擴張實體身分標籤（維持 Core 6）。
- 嚴禁只提問題不提解決方案。

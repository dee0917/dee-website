# 快速開始：內容結構化管理

## 一鍵式操作指南

### 第一次使用？這樣做：

#### 步驟 1：進入結構化管理
1. 登入後台：`http://localhost:5173/admin`
2. 點擊側邊欄的「結構化管理」

#### 步驟 2：查看狀態
頁面會顯示：
- 📊 總文章數：X 篇
- ✅ 已結構化：X 篇
- ⚠️ 待處理：X 篇
- 📈 完成率：XX%

#### 步驟 3：轉換文章
對於每篇待處理文章：
1. 點擊「預覽」查看建議內容
2. 如果滿意，點擊「確認轉換」
3. 或直接點擊「自動轉換」

#### 步驟 4：微調優化
轉換後可以：
1. 點擊「編輯」進入詳細編輯器
2. 調整各個欄位的內容
3. 確保符合你的需求

### 批量轉換大量文章？

如果有多篇文章需要轉換：

```bash
# 執行批量腳本
npx tsx scripts/batch-convert-structure.ts
```

按照提示操作即可一次性轉換所有文章。

## 常見問題

**Q: 轉換後內容不夠好？**
A: 點擊「編輯」進行手動調整，或參考 ID 203 的範例文章

**Q: 如何知道文章是否已結構化？**
A: 在免費教學列表中會顯示「已結構化/未結構化」標記

**Q: 可以批量編輯嗎？**
A: 使用批量腳本或逐個在結構化管理頁面操作

**Q: 轉換會覆蓋原有內容嗎？**
A: 不會，只會添加/更新結構化欄位，原有內容保持不變
